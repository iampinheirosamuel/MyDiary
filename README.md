MyDiary is an online journal where users can pen down their thoughts and feelings.

## Getting Started


### Installation

```
npm install
```

```
npm start
```

## Running the tests

```
npm run test
```

## Built With

* HTML5 and CSS for frontend 
* JavaScript ES6
